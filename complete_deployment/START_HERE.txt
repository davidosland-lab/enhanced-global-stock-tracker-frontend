TROUBLESHOOTING
================================================================================

Issue: "Python is not installed"
Solution: Install Python from https://www.python.org/downloads/
          Make sure to check "Add Python to PATH"

Issue: "ModuleNotFoundError"
Solution: Run INSTALL_DEPENDENCIES.bat again

Issue: "Validation complete: 0/40 passed"
Solution: Check internet connection
          Alpha Vantage API may be rate-limited (wait 1 minute)

Issue: "API usage: 500/500 calls today"
Solution: Daily API limit reached
          Wait until midnight UTC (~10 AM AEST)
          Or upgrade to paid Alpha Vantage tier

Issue: Reports not generating
Solution: Check that stocks were validated (must be > 0)
          Check logs/ directory for error messages
