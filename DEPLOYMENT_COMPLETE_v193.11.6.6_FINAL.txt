**YOUR QUESTION:**
"Is there a single .bat file that installs all dependencies on the first run,
or should I run the 'install first time' .bat followed by 'install complete
v193' .bat?"

**ANSWER:**
You need to run TWO batch files in sequence for complete installation:

1️⃣ FIRST: INSTALL_FIRST_TIME.bat
   Purpose:  Installs 35+ core packages
   Includes: yahooquery, yfinance, pandas, numpy, scipy, scikit-learn,
             tensorflow, torch, transformers (FinBERT), flask, flask-cors,
             plotly, dash, beautifulsoup4, selenium, requests, and more
   Time:     10-15 minutes
   When:     First-time installation or major upgrades

2️⃣ SECOND: INSTALL_COMPLETE_v193.bat
   Purpose:  Installs additional dependencies and configurations
   Includes: Additional optional packages, system configurations
   Time:     5-10 minutes
   When:     After INSTALL_FIRST_TIME.bat completes

TOTAL TIME: 20-30 minutes for complete first-time setup

THEN LAUNCH:
  START_WEB_CONTROL_CENTER.bat → Opens http://localhost:5000

WHY TWO FILES?
  - INSTALL_FIRST_TIME.bat: Core dependencies (35+ packages)
  - INSTALL_COMPLETE_v193.bat: Additional/optional components
  - Separation allows for faster reinstalls and troubleshooting
  - Previous docs incorrectly said "pip install flask flask-cors" (only 2 packages)
  - This caused ModuleNotFoundError for pandas, numpy, tensorflow, etc.
