✅ WHAT THE FIX DOES
================================================================================

Technical summary:

File:    finbert_v4.4.4/models/lstm_predictor.py
Line:    510
Problem: self.model.save(self.model_path)
Fix:     self.model.save(self.model_path, save_format='h5')

Your system has Keras 3.11.3, which requires the save_format='h5' parameter
when saving to .h5 files. Without it, the save fails silently.

The fix adds this parameter plus error handling and Keras 2.x fallback.
