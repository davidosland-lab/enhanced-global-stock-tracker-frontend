TROUBLESHOOTING
================================================================================

ISSUE: "ERROR: overnight_pipeline.py not found!"
  → You're in the wrong directory
  → Solution: cd C:\Users\david\AATelS\TELEGRAM_PHASE8_PATCH
  → Then run: runpatch.bat

ISSUE: "Phase 8 already exists"
  → This means the patch is already installed!
  → No action needed, you're good to go

ISSUE: Patch installed but no Telegram message
  → Check the pipeline log for "PHASE 8: TELEGRAM NOTIFICATIONS"
  → If Phase 8 appears but no message, it's a config issue:
     - Run: python check_telegram_setup.py
     - Verify: config\intraday_rescan_config.json
     - Check: notifications.telegram section has bot_token and chat_id

ISSUE: "Email notification failed: 'bool' object is not callable"
  → This is a Phase 7 issue, not Phase 8
  → Phase 8 should still execute after this error
  → Check if you see "PHASE 8" in the log after the Phase 7 error

ROLLBACK (if needed):
  If something goes wrong, restore backups:
    cd C:\Users\david\AATelS\models\screening
    copy overnight_pipeline.py.backup overnight_pipeline.py /Y
    copy us_overnight_pipeline.py.backup us_overnight_pipeline.py /Y
