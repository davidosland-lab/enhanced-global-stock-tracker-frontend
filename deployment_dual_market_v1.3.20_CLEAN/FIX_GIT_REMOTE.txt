================================================================================
FIX GIT REMOTE - Setup Instructions
================================================================================

ERROR YOU'RE SEEING:
    fatal: 'origin' does not appear to be a git repository
    fatal: Could not read from remote repository.

THE PROBLEM:
    Your local Git repository doesn't have the 'origin' remote configured.
    This means Git doesn't know where to pull updates from.

================================================================================
SOLUTION 1: Add Git Remote (Then Pull Updates)
================================================================================

Step 1: Check current remotes
    cd C:\Users\david\AATelS
    git remote -v

    If empty or missing 'origin', continue to Step 2.

Step 2: Add the remote
    cd C:\Users\david\AATelS
    git remote add origin https://github.com/davidosland-lab/enhanced-global-stock-tracker-frontend.git

Step 3: Verify it was added
    git remote -v

    Should show:
    origin  https://github.com/davidosland-lab/enhanced-global-stock-tracker-frontend.git (fetch)
    origin  https://github.com/davidosland-lab/enhanced-global-stock-tracker-frontend.git (push)

Step 4: Pull the latest updates
    git pull origin finbert-v4.0-development

Step 5: If you get merge conflicts or errors
    git pull origin finbert-v4.0-development --rebase

    OR if you want to keep remote version:
    git fetch origin finbert-v4.0-development
    git reset --hard origin/finbert-v4.0-development
