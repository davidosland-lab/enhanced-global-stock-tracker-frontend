INTEGRATION FLOW:
================

Morning Pipeline Run:
--------------------
1. Overnight pipeline executes (e.g., UK pipeline at 7:00 AM GMT)
2. ftse_proxy_realtime.py computes gap prediction
   - Uses US market closes (S&P, NASDAQ, DOW)
   - Factors in Europe markets (DAX, CAC)
   - Applies VIX confidence adjustment
   - Result: Gap -0.24%, Confidence 70%, Direction NEUTRAL

3. Gap saved in morning_report.json:
   {
     "market_sentiment": {
       "sentiment_score": 29.2,
       "gap_prediction": {
         "predicted_gap_pct": -0.24,
         "confidence": 0.70,
         "direction": "NEUTRAL",
         "method": "REALTIME_MARKET_CLOSES"
       }
     }
   }

Dashboard Display:
-----------------
4. Dashboard loads morning reports via _load_gap_predictions()
5. Extracts gap_prediction from each market report
6. Displays in market status panel with color coding
7. Updates every time dashboard refreshes (automatic)

Paper Trading:
-------------
8. Coordinator loads reports via _load_overnight_reports()
9. Logs gap predictions to console:
   "[OK] Loaded UK morning report - gap -0.24% (NEUTRAL, 70% conf)"
10. Trader sees predictions at startup
11. Can optionally use analyze_pre_market_gaps() for automated trading

TECHNICAL IMPLEMENTATION:
=========================

Dashboard Changes (core/unified_trading_dashboard.py):
------------------------------------------------------

1. New Helper Function:
   ```python
   def _load_gap_predictions():
       """Load gap predictions from overnight pipeline reports"""
       predictions = {}
       report_base = Path(__file__).parent.parent / 'reports' / 'screening'
       
       for market in ['au', 'uk', 'us']:
           report_path = report_base / f'{market}_morning_report.json'
           if report_path.exists():
               with open(report_path, 'r') as f:
                   report = json.load(f)
               market_sentiment = report.get('market_sentiment', {})
               gap_prediction = market_sentiment.get('gap_prediction', {})
               
               if gap_prediction and 'predicted_gap_pct' in gap_prediction:
                   predictions[market] = {
                       'predicted_gap_pct': gap_prediction['predicted_gap_pct'],
                       'confidence': gap_prediction.get('confidence', 0),
                       'direction': gap_prediction.get('direction', 'NEUTRAL'),
                       'available': True
                   }
       
       return predictions
   ```

TROUBLESHOOTING:
===============

Issue: Gap predictions not showing in dashboard
Solution: 
  1. Check if morning reports exist: ls reports/screening/*_morning_report.json
  2. Run overnight pipeline to generate reports
  3. Verify gap_prediction in report: cat reports/screening/uk_morning_report.json | grep gap_prediction

Issue: Paper trading not logging gaps
Solution:
  1. Check if reports are being loaded (look for "[OK] Loaded" messages)
  2. Verify gap_prediction exists in report JSON
  3. Check log level is INFO or DEBUG

Issue: Dashboard shows old gap predictions
Solution:
  1. Dashboard auto-reloads every interval
  2. Or manually refresh browser (F5)
  3. Check report age_hours (should be < 24h)
