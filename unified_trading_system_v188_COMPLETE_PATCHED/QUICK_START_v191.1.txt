───────────────────────────────────────────────────────────────────────────
🔄 UPGRADING FROM v190
───────────────────────────────────────────────────────────────────────────

STEP 1: Stop Dashboard
  Press Ctrl+C in the terminal

STEP 2: Backup Current Installation (Optional but recommended)
  Windows:  xcopy /E /I . ..\v190_backup
  Linux/Mac: cp -r . ../v190_backup

STEP 3: Extract v191.1 Package
  Extract unified_trading_system_v191.1_COMPLETE.zip to your current directory
  (This will update the necessary files)

STEP 4: Clear Python Cache (CRITICAL!)
  Windows:
    for /d /r . %d in (__pycache__) do @if exist "%d" rd /s /q "%d"
    del /s /q *.pyc
  
  Linux/Mac:
    find . -type d -name __pycache__ -exec rm -rf {} +
    find . -type f -name "*.pyc" -delete

STEP 5: Run Diagnostic
  python DEBUG_UK_STOCKS.py

STEP 6: Restart Dashboard
  python start.py

STEP 7: Verify Fix
  - Open http://localhost:8050
  - Wait 2-3 minutes
  - Check BP.L and LGEN.L positions
  - They should now show price changes!

⚠️ COMMON ISSUES
───────────────────────────────────────────────────────────────────────────

PROBLEM: UK stocks still frozen
SOLUTION: 
  1. Run python DEBUG_UK_STOCKS.py
  2. Clear Python cache (see Step 4 above)
  3. Restart dashboard
  4. Wait 2-3 minutes for next update cycle

PROBLEM: Confidence slider shows 65%
SOLUTION:
  1. Clear browser cache (Ctrl+Shift+Delete)
  2. Clear Python cache
  3. Hard refresh page (Ctrl+F5)
  4. Restart dashboard

PROBLEM: "Module not found" errors
SOLUTION:
  pip install -r requirements.txt --upgrade

PROBLEM: Dashboard won't start
SOLUTION:
  1. Check if port 8050 is already in use
  2. Try: python start.py --port 8051
  3. Check logs/unified_trading_system.log
