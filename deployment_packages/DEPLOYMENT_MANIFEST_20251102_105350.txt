DEPLOYMENT INSTRUCTIONS\n
================================================================================\n

1. Extract package to Windows machine
   Example: C:\FinBERT_v4.0\

2. Run installation script as Administrator
   Location: scripts\INSTALL_WINDOWS11.bat
   Right-click → 'Run as Administrator'

3. Choose FULL installation (recommended)
   Press: 1

4. Start application
   Double-click: START_FINBERT_V4.bat

5. Open browser
   Navigate to: http://127.0.0.1:5001

For detailed instructions, see DEPLOYMENT_INSTRUCTIONS.md
