üìñ USAGE GUIDE
================================================================================

Basic Commands:
  python run_screening.py --market us          Run US only
  python run_screening.py --market asx         Run ASX only
  python run_screening.py --market both        Run both (sequential)
  python run_screening.py --market both --parallel   Run both (parallel)

Advanced Options:
  --stocks NUM              Stocks per sector (default: 30)
  --sectors "Sector1,..."   Specific sectors only
  --parallel                Enable parallel execution

Examples:
  # Quick test with 10 stocks
  python run_screening.py --market both --stocks 10

  # US Technology and Healthcare only
  python run_screening.py --market us --sectors "Technology,Healthcare"

  # Full production run (parallel)
  python run_screening.py --market both --parallel
