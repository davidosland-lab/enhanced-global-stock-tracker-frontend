OPTIONAL PACKAGES (FOR BETTER PERFORMANCE)
================================================================================

The system works without these packages (graceful degradation):

OPTIONAL: hmmlearn
  Purpose: HMM-based regime detection (better than GMM)
  Without: Uses Gaussian Mixture Model fallback
  To enable: Uncomment in requirements.txt, re-run installer

OPTIONAL: arch
  Purpose: GARCH volatility forecasting (better than EWMA)
  Without: Uses Exponentially Weighted Moving Average fallback
  To enable: Uncomment in requirements.txt, re-run installer

OPTIONAL: xgboost
  Purpose: XGBoost ensemble models
  Without: Ensemble predictions disabled
  To enable: Uncomment in requirements.txt, re-run installer

To install optional packages:
  1. Edit requirements.txt
  2. Remove the "#" from the optional package lines
  3. Windows: Run INSTALL.bat again
  4. Linux/Mac: Run ./install.sh again
