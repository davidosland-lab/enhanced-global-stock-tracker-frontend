üîß TROUBLESHOOTING
--------------------------------------------------------------------------------

Server Won't Start:
‚Üí Check Python installed: python --version
‚Üí Install dependencies: pip install -r requirements.txt
‚Üí Kill existing process: lsof -ti:5001 | xargs kill -9

Paper Trading Issues:
‚Üí Check browser console (F12) for errors
‚Üí Test API: curl http://localhost:5001/api/trading/account
‚Üí Verify models/trading/ directory exists

Low Predictions:
‚Üí Train LSTM model for the stock
‚Üí Check market is open (after-hours = lower confidence)
‚Üí Verify stock symbol is valid
