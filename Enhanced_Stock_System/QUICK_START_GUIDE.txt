TROUBLESHOOTING:
---------------
If server doesn't start:
- Make sure Python 3.8+ is installed
- Run: pip install flask yfinance pandas numpy scikit-learn requests
- Check port 8000 is free: netstat -an | findstr :8000

If no data loads:
- Check internet connection
- Try switching data source in the interface
- Yahoo Finance may have temporary issues, system will auto-switch to Alpha Vantage
