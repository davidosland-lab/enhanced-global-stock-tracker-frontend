## 🛠️ Troubleshooting (Common Issues)

### Issue: "Python is not installed"
**Solution:** Download and install Python 3.8+ from https://www.python.org/
- ⚠️ Check "Add Python to PATH" during installation

### Issue: Dashboard won't start
**Solution:** 
1. Press Ctrl+C to stop
2. Run `start.bat` again
3. If port 8050 busy, close other Dash apps

### Issue: "Config file not found"
**Solution:** 
- File will be created automatically
- Check `config/live_trading_config.json` exists after first run

### Issue: Trades still blocked at 65%
**Solution:**
1. Stop dashboard (Ctrl+C)
2. Delete cache:
   ```cmd
   rmdir /s /q core\__pycache__
   rmdir /s /q ml_pipeline\__pycache__
   ```
3. Restart with `start.bat`
