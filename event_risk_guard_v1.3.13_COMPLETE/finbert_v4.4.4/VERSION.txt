QUICK START
================================================================================

1. Extract ZIP file to desired location
   Example: C:\FinBERT_v4.4\

2. Run INSTALL.bat as Administrator
   - Checks Python installation
   - Installs all dependencies
   - Configures environment

3. Run START_FINBERT.bat
   - Starts Flask server on port 5001
   - Access at: http://localhost:5001

4. Test with Australian stock
   - Navigate to: http://localhost:5001/api/sentiment/CBA.AX
   - Should show 12+ articles with Australian context

SUPPORT & TROUBLESHOOTING
================================================================================

Common Issues:
1. Port 5001 already in use
   ‚Üí Run: netstat -ano | findstr :5001
   ‚Üí Kill process or use different port

2. Flask-CORS errors
   ‚Üí Run: FIX_FLASK_CORS.bat

3. Python not found
   ‚Üí Install Python 3.10 from python.org
   ‚Üí Check "Add Python to PATH" during install

4. Dependencies failed
   ‚Üí Run: pip install --upgrade pip
   ‚Üí Run: INSTALL.bat again

5. RBA sources not fetching
   ‚Üí Check internet connection
   ‚Üí Verify Australian stock symbol ends with .AX
   ‚Üí RBA scraping respects 2-second delays

For detailed troubleshooting, see: TROUBLESHOOTING_FLASK_CORS.md
