## 🏗️ Architecture

### Integration Points

```

│                     Trading System                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │

│           │                           │                    │
│           └───────────┬───────────────┘                    │
│                       │                                    │

│    │ Cache   │  │  Rate   │  │ Secure  │                │
│    │ (4h TTL)│  │ Limiter │  │ Config  │                │
│    └─────────┘  └─────────┘  └────┬────┘                │
│                                    │                      │

│                                                           │
└───────────────────────────────────────────────────────────┘
                              │
                     ┌────────▼──────────┐
                     │   EODHD API       │
                     │   (20 calls/day)  │
                     └───────────────────┘
```

### Data Flow

1. **Pipeline Request** → SPI Monitor or FTSE Predictor needs gap prediction
2. **EODHD Client** → Checks cache (4-hour TTL)
3. **Cache Miss** → Checks rate limit (2/20 daily)
4. **API Call** → Fetches real-time futures data
5. **Response** → Caches result, records API call
6. **Prediction** → Returns 95% confidence gap prediction
7. **Fallback** → If EODHD fails, cascade to next method

## 📞 Support & Troubleshooting

### Common Issues

**Issue 1: "EODHD API key not configured"**
- Check .env file exists: `dir .env` (Windows) or `ls .env` (Linux)
- Verify key is not placeholder: Open .env and check it's your actual key
- Restart Python after setting environment variable

**Issue 2: "Rate limit exceeded"**
- Check state file: `type state\eodhd_rate_limit.json` (Windows)
- Wait until midnight UTC (rate limit resets)
- System will automatically use fallback methods

**Issue 3: "Symbol not found"**
- Verify EODHD account is active
- Check account tier includes futures data
- Try alternative symbols in EODHD dashboard
