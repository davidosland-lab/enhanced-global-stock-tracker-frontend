QUICK START - TRAINING MODELS
================================================================================

1. Extract ZIP to desired location
   Example: C:\FinBERT_v4.4.2\

2. Run INSTALL.bat as Administrator
   - Installs Python dependencies
   - Sets up virtual environment

3. Run TRAIN_LSTM_OVERNIGHT.bat
   - Trains LSTM models for top 10 stocks
   - Takes 1-2 hours (best run overnight)
   - Creates models/lstm_SYMBOL_model.keras files

4. Run START_FINBERT.bat
   - Starts server with trained models
   - Access at http://localhost:5001

5. Test improved predictions
   curl http://localhost:5001/api/stock/AAPL
   Should show: "lstm_trained": true, accuracy: 93%

================================================================================
TROUBLESHOOTING
================================================================================

Issue: TensorFlow not found during training
Fix:   pip install tensorflow
       (or run INSTALL.bat again)

Issue: Training interrupted/crashed
Fix:   Just run TRAIN_LSTM_OVERNIGHT.bat again
       Successfully trained models are preserved

Issue: Models not loading after training
Fix:   Check models directory for .keras files
       Restart server: START_FINBERT.bat

Issue: Lower accuracy than expected
Fix:   May need more epochs (edit train_lstm_batch.py)
       Some stocks harder to predict (high volatility)
       Retrain after major market events

Issue: Training takes too long
Fix:   Normal for CPU (10-15 min per stock)
       GPU would be 3-5 min per stock
       Can train individual stocks: python models/train_lstm.py --symbol AAPL
