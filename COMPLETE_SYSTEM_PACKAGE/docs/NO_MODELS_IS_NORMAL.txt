================================================================================
UNDERSTANDING TRAINING TIME
================================================================================

WHY 5-15 MINUTES PER STOCK?
----------------------------
For each stock, the system:
1. Fetches 3 months of historical data (~60 days)
2. Prepares training sequences (60-day windows)
3. Trains 3-layer neural network (50 epochs)
4. Validates on 20% holdout data
5. Saves model weights to .h5 file

Each epoch processes all training data once.
50 epochs = 50 complete passes through the data.

WHAT AFFECTS TRAINING TIME:
----------------------------
- Stock volatility (more patterns = longer training)
- Amount of data available
- CPU speed (no GPU required)
- System RAM (2-4GB per stock)

TYPICAL TIMES:
--------------
Small/stable stock: 5-8 minutes
Medium stock: 8-12 minutes
Large/volatile stock: 12-15 minutes

================================================================================
FREQUENTLY ASKED QUESTIONS
================================================================================

Q: Why do I need to train models?
A: LSTM neural networks need to learn patterns from historical data.
   Pre-trained models wouldn't know about specific ASX stocks.

Q: Can I use the system without training?
A: Yes! System uses fallback predictions (trend + technical + sentiment).
   But accuracy improves significantly with trained LSTM models.

Q: How often should I retrain?
A: Models become "stale" after 7 days.
   Recommendation: Train priority stocks weekly or daily.

Q: Which stocks should I train first?
A: System automatically prioritizes by:
   - Highest opportunity scores
   - Largest market cap (CBA, BHP, RIO, WES, CSL)
   - Most liquid stocks
   - Your portfolio holdings (if specified)

Q: What if training fails for a stock?
A: System will skip it and use fallback predictions.
   Common reasons: Insufficient data, delisted stock, network timeout.

Q: Do I need to train all 240 stocks?
A: No! Start with 3-10 stocks you care about most.
   The system works fine with partial training.
