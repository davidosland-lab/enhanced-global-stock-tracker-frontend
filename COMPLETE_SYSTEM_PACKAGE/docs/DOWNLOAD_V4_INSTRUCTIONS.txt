üêõ TROUBLESHOOTING
================================================================================

Problem: "Python not found"
Solution: Install Python 3.8+ from python.org
          Add to system PATH during installation

Problem: "Module not found"
Solution: Run INSTALL_V4.bat
          Or: pip install -r requirements.txt

Problem: "Port 5001 in use"
Solution: Edit config_dev.py
          Change: PORT = 5002

Problem: "No data for stock"
Solution: Check spelling (case-sensitive)
          For ASX: Include .AX suffix
          Example: CBA.AX not CBA

Problem: "Server won't start"
Solution: Check Python version: python --version
          Reinstall dependencies: INSTALL_V4.bat
          Check firewall settings
