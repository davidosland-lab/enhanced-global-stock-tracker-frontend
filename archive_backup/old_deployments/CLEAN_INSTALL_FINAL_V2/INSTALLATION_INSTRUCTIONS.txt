QUICK INSTALLATION (Windows)
================================================================================

1. EXTRACT FILES
   - Extract CLEAN_INSTALL_FINAL_V2.zip to C:\StockTracker
   - Ensure all files are extracted (15 files total)

2. INSTALL PYTHON (if needed)
   - Download Python 3.8+ from python.org
   - During install, CHECK "Add Python to PATH"
   - Verify: Open CMD and type: python --version

3. INSTALL DEPENDENCIES
   - Open Command Prompt (CMD)
   - Navigate to folder: cd C:\StockTracker\CLEAN_INSTALL_FINAL_V2
   - Run: INSTALL.bat
   - This installs all required Python packages

4. START SERVICES
   - In the same folder, run: START_ALL_SERVICES.bat
   - This starts all three services (Frontend, Backend, ML)
   - Three console windows will open - keep them running

5. ACCESS APPLICATION
   - Open web browser
   - Go to: http://localhost:8000
   - You should see the Stock Tracker Pro dashboard

6. VERIFY INSTALLATION
   - Run: python TEST_SYSTEM.py
   - All services should show as "ONLINE"
   - CBA.AX should show ~$170 (real price)

================================================================================
TROUBLESHOOTING
================================================================================

IF SERVICES WON'T START:
- Check Python is installed: python --version
- Install packages manually: pip install -r requirements.txt
- Check ports aren't in use: netstat -ano | findstr :8000

IF "BACKEND DISCONNECTED":
- Verify backend.py is running (check console window)
- Try refreshing browser (Ctrl+F5)
- Check Windows Firewall isn't blocking

IF FINBERT NOT WORKING:
- Install manually: pip install transformers torch
- First run downloads model (~400MB)
- Be patient on first document analysis

IF WRONG PRICES/DATA:
- Check internet connection
- Verify Yahoo Finance is accessible
- Clear browser cache (Ctrl+F5)
