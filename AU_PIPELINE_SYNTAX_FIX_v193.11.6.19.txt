   ```

 WHAT THIS MODULE DOES
================================================================================

**ai_market_impact_analyzer.py:**
- Uses GPT (OpenAI) to analyze major news events
- Evaluates market impact (bullish/bearish/neutral)
- Assigns severity scores (0-100)
- Provides confidence ratings
- Parses JSON responses from AI

**JSON Extraction Process:**
1. Send news articles to GPT
2. GPT responds with JSON wrapped in markdown:
   ```json
   {
     "impact_score": 2.5,
     "severity": "MODERATE"
   }
   ```
3. Python extracts JSON by finding ` ```json ` and ` ``` ` markers
4. The bug used wrong markers (`''''`) causing syntax error
