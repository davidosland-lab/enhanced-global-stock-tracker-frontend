┌────────────────────────────────────────────────────────────┐

│   finbert_v4.4.4/models/ = "Just a directory"              │
│                                                            │
│ When code does:                                            │
│   from .finbert_sentiment import X                         │
│                                                            │
│ Python thinks:                                             │
│   "Current module is not in a package!"                    │
