================================================================================
FinBERT v4.4 - QUICK START GUIDE
================================================================================

STEP 1: INSTALL
--------------------------------------------------------------------------------
1. Extract the ZIP file to a folder (e.g., C:\FinBERT)
2. Double-click: INSTALL.bat
3. Wait for installation to complete (5-10 minutes)
4. You should see "Installation Complete!" message

STEP 2: START THE SERVER
--------------------------------------------------------------------------------
1. Double-click: START_FINBERT.bat
2. Wait for "Running on http://127.0.0.1:5002" message
3. Keep this window open while using FinBERT

STEP 3: OPEN THE WEB INTERFACE
--------------------------------------------------------------------------------
1. Open your web browser
2. Navigate to: http://localhost:5002
3. You should see the FinBERT prediction interface

================================================================================
USING THE 5 MAIN FEATURES
================================================================================

1. GET PREDICTIONS
--------------------------------------------------------------------------------
   - Enter a stock symbol (e.g., AAPL, TSLA, SPY)
   - Click "Get Prediction"
   - View the 4-model ensemble prediction
   - See technical indicators and sentiment analysis

2. PAPER TRADING
--------------------------------------------------------------------------------
   - Click "Paper Trading" button at top
   - Your virtual account starts with $10,000
   - Enter symbol, quantity, order type
   - Click "BUY" or "SELL"
   - Track positions and P&L in real-time

3. BACKTEST STRATEGY
--------------------------------------------------------------------------------
   - Click "Backtest" button
   - Enter symbol and date range (e.g., 2023-01-01 to 2024-01-01)
   - Set initial capital (default: $10,000)
   - Click "Run Backtest"
   - View equity curve, metrics, and trade log

4. PORTFOLIO BACKTEST
--------------------------------------------------------------------------------
   - Click "Portfolio Backtest" button
   - Enter multiple symbols (comma-separated: AAPL,MSFT,GOOGL)
   - Set date range and initial capital
   - Click "Run Portfolio Backtest"
   - View portfolio performance, stock breakdown, correlation matrix

5. OPTIMIZE PARAMETERS
--------------------------------------------------------------------------------
   - Click "Optimize" button
   - Enter symbol and date range
   - Choose optimization method (grid/random)
   - Click "Run Optimization"
   - View top performing configurations

6. PREDICTION HISTORY
--------------------------------------------------------------------------------
   - Click "Prediction History" button
   - Select symbol and date range
   - View all historical predictions
   - See accuracy comparison with actual outcomes

================================================================================
MULTI-TIMEZONE SUPPORT
================================================================================

FinBERT supports predictions for 3 major markets:

US STOCKS (e.g., AAPL, TSLA, MSFT)
   - Market Hours: 9:30 AM - 4:00 PM Eastern Time
   - Predictions locked at: 8:00 AM Eastern Time (90 min before open)
   - Auto-validation at: 4:15 PM Eastern Time

AUSTRALIAN STOCKS (e.g., CBA.AX, BHP.AX)
   - Market Hours: 10:00 AM - 4:00 PM Sydney Time
   - Predictions locked at: 8:30 AM Sydney Time
   - Auto-validation at: 4:15 PM Sydney Time

UK STOCKS (e.g., BARC.L, BP.L)
   - Market Hours: 8:00 AM - 4:30 PM London Time
   - Predictions locked at: 6:30 AM London Time
   - Auto-validation at: 4:45 PM London Time

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "ModuleNotFoundError: No module named 'flask_cors'"
Solution: Run FIX_FLASK_CORS.bat

Problem: Server won't start
Solution: 
   1. Run VERIFY_INSTALL.bat to check setup
   2. Make sure no other program is using port 5002
   3. Check that Python 3.8+ is installed

Problem: Can't get predictions
Solution:
   1. Check internet connection (needs to fetch market data)
   2. Use valid stock symbols (check Yahoo Finance)
   3. Make sure market data is available for the symbol

Problem: Installation takes too long
Solution:
   - TensorFlow and PyTorch are large packages (1-2 GB)
   - Installation can take 10-20 minutes on slow connections
   - This is normal - just wait for completion

================================================================================
ADVANCED USAGE
================================================================================

Running on a different port:
   1. Edit config_dev.py
   2. Change: app.run(port=5002) to app.run(port=YOUR_PORT)
   3. Restart server

Using with real-time data:
   - FinBERT uses yfinance for market data
   - Data is typically delayed 15-20 minutes
   - For real-time data, consider upgrading to paid data feeds

Training custom LSTM models:
   - See the training component (to be integrated next phase)
   - Requires TensorFlow/Keras installation
   - Recommended: 16GB+ RAM for training
