================================================================================
🔒 RISK MANAGEMENT
================================================================================

MACRO RISK GATES (v193.10)
---------------------------
1. World Risk Gate
   - Block if World Risk ≥ 80/100
   - Reduce position 50% if ≥ 60/100

2. US Market Gate
   - Block if S&P ≤ -1.5%
   - Reduce position 25% if ≤ -0.75%

3. VIX Gate
   - Block if VIX ≥ 30 and confidence < 70%
   - Reduce position 25% if VIX ≥ 25

4. Financial Sector Gate
   - Block if World Risk ≥ 60
   - Block if US Market ≤ -1.0%

GAP PREDICTION GATES
--------------------
1. Confidence Gate
   - Require ≥ 60% confidence for entry

2. Magnitude Gate
   - Require gap ≥ 0.3% for action

3. Sentiment Alignment
   - Dampen 15-25% if gap/sentiment disagree

4. World Risk Dampening
   - Reduce 50% if World Risk ≥ 85
   - Reduce 25% if World Risk ≥ 80

POSITION SIZING FORMULA
-----------------------
Position Size = Base × Confidence Factor × Gap Factor × Risk Multiplier

Confidence Factor:
- 60% conf = 0.6x
- 70% conf = 0.85x
- 80% conf = 1.0x
- 90% conf = 1.2x

Gap Factor:
- 0.3% gap = 0.7x
- 1.0% gap = 1.0x
- 2.0% gap = 1.3x

Risk Multiplier:
- World Risk < 60: 1.0x
- World Risk 60-80: 0.75x
- World Risk 80-85: 0.5x
- World Risk ≥ 85: 0.25x

================================================================================
🐛 TROUBLESHOOTING
================================================================================

ISSUE: Dashboard won't start
SOLUTION:
1. Check Python version: python --version (need 3.10+)
2. Reinstall packages: INSTALL_FIRST_TIME.bat
3. Check port 8050: netstat -ano | findstr :8050
4. Kill process if needed: taskkill /PID <pid> /F

ISSUE: Gap predictions showing 0.0%
SOLUTION:
1. Run overnight pipelines first (RUN_AU_PIPELINE.bat, etc.)
2. Check reports exist: dir reports\screening\*_morning_report.json
3. Check report age (must be < 24 hours)

ISSUE: US gap prediction failing
SOLUTION:
1. Check futures data: python pipelines/models/screening/spx_proxy_realtime.py
2. Verify internet connection (needs Yahoo Finance)
3. Check log: logs/screening/us/us_overnight_pipeline.log

ISSUE: No trading signals in dashboard
SOLUTION:
1. Lower confidence threshold (Settings panel, default 48%)
2. Run overnight pipelines to generate signals
3. Click "Load Top 50" button to import from reports

ISSUE: ModuleNotFoundError
SOLUTION:
1. Run INSTALL_FIRST_TIME.bat (installs all 35+ packages)
2. Do NOT use "pip install flask flask-cors" only (insufficient)
3. Check: pip list | findstr "pandas numpy dash"

ISSUE: UnicodeEncodeError on Windows
SOLUTION:
1. Already fixed in v193.11.5+
2. All Unicode symbols replaced with ASCII
3. Update to latest version if you see this error
