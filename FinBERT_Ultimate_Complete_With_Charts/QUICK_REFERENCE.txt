================================================================
   FINBERT ULTIMATE TRADING SYSTEM - QUICK REFERENCE
================================================================

INSTALLATION (One Time Only)
----------------------------
1. Run: INSTALL.bat
2. Follow prompts (takes 2-5 minutes)

DAILY USE
---------
1. Run: START.bat
2. Charts open automatically
3. Enter stock symbol
4. Click "Analyze"

KEY FEATURES
------------
• Next-Day Price Prediction    - Tomorrow's expected price
• 5-10 Day Target              - Medium-term price target
• Sentiment Score              - -1 (bearish) to +1 (bullish)
• Technical Indicators         - RSI, MACD, Bollinger, SMA
• Economic Dashboard           - VIX, Treasury, Dollar, Gold, Oil
• News Sentiment              - Latest news with AI analysis

SUPPORTED SYMBOLS
-----------------
US Stocks:     AAPL, MSFT, GOOGL, TSLA, etc.
ETFs:          SPY, QQQ, DIA, IWM
Crypto:        BTC-USD, ETH-USD
International: .AX (Australia), .L (UK), .DE (Germany)

CHART CONTROLS
--------------
• Zoom:        Mouse wheel or pinch
• Pan:         Click and drag
• Reset:       Double-click
• Indicators:  Click badges to toggle

TIMEFRAMES
----------
1D  = 1 Day (intraday)
5D  = 5 Days (week)
1M  = 1 Month
3M  = 3 Months
6M  = 6 Months
1Y  = 1 Year

API ENDPOINTS
-------------
http://localhost:5000/api/stock/{symbol}      - Get stock data
http://localhost:5000/api/predict/{symbol}    - Get prediction
http://localhost:5000/api/train              - Train model
http://localhost:5000/api/historical/{symbol} - Historical data
http://localhost:5000/api/news/{symbol}      - News & sentiment

PREDICTION CONFIDENCE
---------------------
80-100% = Strong signal
60-80%  = Moderate confidence  
40-60%  = Weak signal
0-40%   = Very uncertain

TECHNICAL INDICATORS
--------------------
RSI > 70    = Overbought (may fall)
RSI < 30    = Oversold (may rise)
MACD Cross  = Trend change signal
Bollinger   = Volatility measure
SMA Cross   = Trend confirmation

TROUBLESHOOTING
---------------
Problem: Charts not loading
→ Solution: Wait 5 seconds, refresh browser

Problem: No predictions
→ Solution: First prediction takes 30-60 seconds

Problem: Installation error
→ Solution: Run: pip install "numpy>=1.26.0"

Problem: Server won't start
→ Solution: Check Python 3.10+ installed

TIPS FOR SUCCESS
----------------
1. Train models weekly for best accuracy
2. Use 6-month periods for predictions
3. Check multiple timeframes
4. Monitor sentiment changes
5. Combine with your own analysis
6. Never trade solely on predictions
7. Use stop-losses based on ATR
8. Paper trade first to test
