ROOT CAUSE:
-----------
The code checked market hours and logged warnings, but DID NOT actually prevent
trading on closed markets:

  BAD CODE (paper_trading_coordinator.py line ~2573):
  ----------------------------------------------------
  # Check market hours for all symbols
  if MARKET_CALENDAR_AVAILABLE and market_calendar:
      closed_symbols = []
      for symbol in self.symbols:
          can_trade, reason = market_calendar.can_trade_symbol(symbol)
          if not can_trade:
              closed_symbols.append(f"{symbol} ({reason})")
      
      if closed_symbols:
          logger.warning(f"[CALENDAR] Some markets are closed:")
          for msg in closed_symbols:
              logger.warning(f"   {msg}")
          # Continue with open markets only  ← COMMENT, NO ACTUAL CODE!
  
  # Later in same function (line ~2646)...
  for symbol in self.symbols:  ← STILL CHECKS ALL SYMBOLS!
      if symbol in self.positions:
          continue
      should_enter, confidence, signal = self.evaluate_entry(symbol)
      if should_enter:
          logger.info(f"[OK] Entry signal for {symbol}")
          self.enter_position(symbol, signal)  ← NO MARKET CHECK HERE EITHER!

RESULT: System logged "NAB.AX (After-hours)" but still evaluated it for entry
and executed trades on closed markets.

THE FIX:
--------
Three-layer protection against after-hours trading:

1. FILTER SYMBOLS BY MARKET STATUS (line ~2573):
   ---------------------------------------------
   open_symbols = []
   closed_symbols = []
   
   if MARKET_CALENDAR_AVAILABLE and market_calendar:
       for symbol in self.symbols:
           can_trade, reason = market_calendar.can_trade_symbol(symbol)
           if can_trade:
               open_symbols.append(symbol)  # Only tradeable markets
           else:
               closed_symbols.append(f"{symbol} ({reason})")
       
       logger.info(f"[CALENDAR] Trading only on open markets: {len(open_symbols)} of {len(self.symbols)} symbols")
   else:
       open_symbols = self.symbols.copy()

2. SCAN ONLY OPEN MARKETS (line ~2646):
   -------------------------------------
   for symbol in open_symbols:  # ← Changed from self.symbols!
       if symbol in self.positions:
           continue
       should_enter, confidence, signal = self.evaluate_entry(symbol)
       if should_enter:
           self.enter_position(symbol, signal)

3. SAFETY CHECK IN enter_position() (line ~2010):
   -----------------------------------------------
   def enter_position(self, symbol: str, signal: Dict) -> bool:
       try:
           # MARKET HOURS CHECK (NEW!)
           if not signal.get('type') == 'MANUAL' and MARKET_CALENDAR_AVAILABLE and market_calendar:
               can_trade, reason = market_calendar.can_trade_symbol(symbol)
               if not can_trade:
                   logger.warning(f"{symbol}: TRADE BLOCKED - Market closed ({reason})")
                   return False
           
           # ... rest of function

WHAT ABOUT MANUAL TRADES?
--------------------------
Manual trades (triggered via dashboard UI) can OVERRIDE the market hours check.
This is intentional for testing and special cases:

  signal = {'type': 'MANUAL', 'shares': 100, ...}
  enter_position('NAB.AX', signal)  ← Will execute even if market closed

Automatic trades (from ML signals) are always blocked on closed markets.
