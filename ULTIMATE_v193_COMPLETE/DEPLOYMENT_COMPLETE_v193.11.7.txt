🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

PROBLEM: "Python not found"
SOLUTION:
  1. Install Python 3.12 from https://www.python.org/downloads/
  2. During installation, CHECK "Add Python to PATH"
  3. Restart Command Prompt
  4. Run: python --version

PROBLEM: "pip install failed"
SOLUTIONS:
  1. Check internet connection
  2. Run Command Prompt as Administrator
  3. Disable antivirus temporarily
  4. Try: pip install --upgrade pip
  5. Try PyTorch separately:
     pip install torch==2.6.0 --index-url https://download.pytorch.org/whl/cpu --no-cache-dir

PROBLEM: "Trading loop stopped" ← THIS IS NOW FIXED!
STATUS: ✅ FIXED IN v193.11.7
EXPLANATION:
  - System now has comprehensive exception handling
  - Loop continues automatically after errors
  - Check logs for "[WARN] Trading cycle failed but loop will continue"
  - No manual restart needed

PROBLEM: "Dashboard shows old data"
SOLUTION:
  1. Stop dashboard (Ctrl+C)
  2. Run fresh pipeline (START.bat → Option 5/6/7)
  3. Restart dashboard (START.bat → Option 3)
  4. Clear browser cache (Ctrl+Shift+Delete)

PROBLEM: "LSTM failed - not enough data"
SOLUTION:
  1. Run START.bat → Option 8 (Train Individual Stocks)
  2. Enter the symbol (e.g., CBA.AX)
  3. Wait for training (~3 minutes)
  4. Restart dashboard

PROBLEM: "Port already in use"
SOLUTION:
  1. Find process:
     netstat -ano | findstr ":8050"
     netstat -ano | findstr ":5001"
  2. Kill process:
     taskkill /PID <process_id> /F
  3. Restart START.bat
