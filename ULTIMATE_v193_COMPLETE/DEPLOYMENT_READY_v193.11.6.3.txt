🐛 KNOWN ISSUES & SOLUTIONS
================================================================================

Issue: "ModuleNotFoundError: No module named 'flask'"
Solution: pip install flask flask-cors

Issue: Port 5000 already in use
Solution: Edit web_control_center.py, change port on line ~390:
         app.run(host='0.0.0.0', port=5001, debug=False)

Issue: Dashboard iframe not loading
Solution: 
  1. Ensure dashboard is running (check status badge)
  2. Open http://localhost:8050 in separate tab to verify
  3. Wait 30 seconds for dashboard startup

Issue: Logs not streaming
Solution:
  1. Refresh the page (F5)
  2. Check browser console (F12) for errors
  3. Ensure service is actually running

Issue: Can't access from phone/tablet
Solution:
  1. Ensure PC firewall allows port 5000
  2. Verify both devices on same Wi-Fi
  3. Use PC's IPv4 address, not "localhost"
