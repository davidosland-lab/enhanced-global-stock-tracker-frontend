📱 THREE STEPS TO MOBILE ACCESS:

  1. INSTALL NGROK (one-time)
     • Download: https://ngrok.com/download
     • Sign up: https://dashboard.ngrok.com/signup
     • Run: ngrok authtoken YOUR_TOKEN

  2. START MOBILE ACCESS
     • Double-click: START_MOBILE_ACCESS.bat
     • Set username/password when prompted
     • Wait for QR code to appear

  3. CONNECT FROM PHONE
     • Scan QR code with phone camera
     • OR manually visit the https://... URL shown
     • Enter username and password
     • Start trading!

🔧 TROUBLESHOOTING:

  Problem: ngrok not found
  → Install ngrok and add to PATH

  Problem: Can't connect from phone
  → Check dashboard is running at http://localhost:8050
  → Verify firewall allows connections
  → Try incognito/private mode

  Problem: Wrong credentials
  → Check config/mobile_connection_info.txt
  → Password is case-sensitive

  Problem: Tunnel expired
  → Free ngrok tunnels expire after 2 hours inactive
  → Just restart START_MOBILE_ACCESS.bat
