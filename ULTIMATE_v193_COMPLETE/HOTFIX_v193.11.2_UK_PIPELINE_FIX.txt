ISSUE SUMMARY
===============================================================================
The UK overnight pipeline failed to initialize with the error:
  NameError: name '_event_monitor' is not defined

This prevented the UK pipeline from running and generating stock recommendations.

ROOT CAUSE
===============================================================================
File: pipelines/models/screening/world_event_monitor.py
Line: 411

The file had leftover test code at the module level:
  
  if __name__ == '__main__':
      test_world_event_monitor()
  _event_monitor()  # <-- This line caused the error

When the module was imported (not run as main), line 411 executed and tried
to call `_event_monitor()` which was never defined.

SOLUTION IMPLEMENTED
===============================================================================
Removed the erroneous line 411:

Before:
-------
if __name__ == '__main__':
    test_world_event_monitor()
_event_monitor()

After:
------
if __name__ == '__main__':
    test_world_event_monitor()

This ensures the test function only runs when the module is executed directly,
not when it's imported by the UK pipeline.
