QUICK START (3 SIMPLE STEPS)
============================================================

1. INSTALL PYTHON (if not already installed)
   - Download Python 3.7 or newer from https://www.python.org/
   - During installation, CHECK "Add Python to PATH"

2. INSTALL REQUIREMENTS
   - Double-click: INSTALL_REQUIREMENTS.bat
   - Wait for all packages to install

3. START THE APPLICATION
   - Double-click: START_FIXED_SERVICES.bat
   - Open browser to: http://localhost:8000

============================================================

============================================================
TROUBLESHOOTING
============================================================

If services don't start:
1. Make sure Python is installed
2. Run INSTALL_REQUIREMENTS.bat
3. Check Windows Firewall settings
4. Make sure ports 8000, 8002, 8003 are free

If you see 404 errors:
- Stop all services (STOP_ALL.bat)
- Start again (START_FIXED_SERVICES.bat)
- Clear browser cache (Ctrl+Shift+Delete)
