================================================================================
TROUBLESHOOTING
================================================================================

Issue: "UNKNOWN" Regime
  Solution: Run python diagnose_regime.py
           Check internet connection
           Verify yfinance version (pip list | grep yfinance)

Issue: No Models in Web UI
  Solution: Check models/ directory exists
           Re-run pipeline: python run_pipeline.py
           Check logs for errors

Issue: Training Fails
  Solution: Verify TensorFlow installed (pip list | grep tensorflow)
           Check memory (need 4GB+ RAM)
           Clear cache: rm data/stocks_cache.json

Issue: Import Errors
  Solution: Reinstall: pip install -r requirements.txt
           Check Python version: python --version (need 3.8+)
           Verify package structure (all __init__.py present)

Issue: yfinance Errors
  Solution: Upgrade: pip install --upgrade yfinance
           Version should be 0.2.66 or higher
