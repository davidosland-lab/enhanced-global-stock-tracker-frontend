=====================================================
STOCK PREDICTOR PRO - INSTALLATION FIX
=====================================================

IMPORTANT: Extract ALL files to a folder first!
Do NOT run from inside the ZIP file!

STEP 1: EXTRACT THE FILES
-------------------------
1. Right-click the ZIP file
2. Select "Extract All..."
3. Choose a simple location like:
   - C:\StockPredictor
   - Desktop\StockPredictor
   - D:\StockPredictor
   
   NOT in Program Files or System32!

4. Make sure ALL files are extracted:
   - stock_predictor_lite.py
   - PORTABLE_RUN.bat
   - test_simple.py
   - All other files

STEP 2: TEST PYTHON
-------------------
1. Open the extracted folder
2. Double-click: test_simple.py
3. If a window appears saying "Python is working" - you're good!
4. If not, install Python from python.org

STEP 3: RUN THE APPLICATION
----------------------------
Option A: Use PORTABLE_RUN.bat
   - Double-click PORTABLE_RUN.bat
   - This will find Python and run the app

Option B: Direct Python
   - Right-click in the folder
   - Select "Open in Terminal" or "Open Command Prompt here"
   - Type: python stock_predictor_lite.py
   - Press Enter

TROUBLESHOOTING
---------------
Error: "No such file or directory"
Solution: You're running from wrong location. Extract all files first!

Error: "Python not found"
Solution: Install Python 3.9+ from python.org

Error: "Module not found"
Solution: The lite version needs NO modules! Just run it.

QUICK TEST
----------
1. Navigate to where you extracted the files
2. You should see:
   - stock_predictor_lite.py (14 KB)
   - PORTABLE_RUN.bat (3 KB)
   - Other files
3. Double-click PORTABLE_RUN.bat

If this doesn't work:
- Hold Shift + Right-click in the folder
- Select "Open PowerShell window here"
- Type: python stock_predictor_lite.py
- Press Enter

=====================================================
The LITE version requires ONLY Python - no packages!
It will work if Python is installed.
