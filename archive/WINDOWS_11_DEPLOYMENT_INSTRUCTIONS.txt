================================================================================
   WINDOWS 11 INSTALLATION STEPS
================================================================================

STEP 1: SYSTEM REQUIREMENTS
----------------------------
• Windows 11 (also works on Windows 10)
• Python 3.8 or higher installed
• Internet connection for Yahoo Finance data
• Web browser (Chrome, Edge, Firefox recommended)
• Minimum 100 MB free disk space

STEP 2: DOWNLOAD THE PACKAGE
-----------------------------
Download: GSMT_v93_COMPLETE_FIXED_Windows.zip (37 KB)

STEP 3: EXTRACT THE PACKAGE
----------------------------
1. Create a folder: C:\GSMT
2. Extract the ZIP file to C:\GSMT
3. You should see these folders:
   - C:\GSMT\backend\
   - C:\GSMT\frontend\
   - C:\GSMT\*.bat files

STEP 4: INSTALL DEPENDENCIES (FIRST TIME ONLY)
-----------------------------------------------
1. Open Command Prompt or PowerShell as Administrator
2. Navigate to C:\GSMT:
   cd C:\GSMT
3. Run the installation script:
   INSTALL_DEPENDENCIES.bat
4. Wait for all packages to install (about 2-3 minutes)

STEP 5: LAUNCH THE APPLICATION
-------------------------------
1. Double-click START_GSMT.bat
2. Two command windows will open:
   - Backend server (Port 8000)
   - Frontend server (Port 3001)
3. Your browser will automatically open to the dashboard
4. If browser doesn't open, navigate to: http://localhost:3001

================================================================================
   TROUBLESHOOTING
================================================================================

PYTHON NOT FOUND:
-----------------
• Download Python from python.org
• Install Python 3.8 or higher
• Check "Add Python to PATH" during installation
• Restart Command Prompt after installation

PORT ALREADY IN USE:
--------------------
• Run STOP_GSMT.bat to stop services
• Open Task Manager
• End any Python processes
• Try launching again

DATA NOT LOADING:
-----------------
• Check internet connection
• Verify Yahoo Finance is accessible
• Check firewall settings
• Look at console windows for errors

MODULE LINKS NOT WORKING:
-------------------------
• Ensure both servers are running (port 8000 and 3001)
• Clear browser cache (Ctrl+Shift+Delete)
• Try a different browser
• Check browser console (F12) for errors

================================================================================
   STOPPING THE APPLICATION
================================================================================

To stop all services:
1. Double-click STOP_GSMT.bat
   OR

   OR
3. Press Ctrl+C in each command window
