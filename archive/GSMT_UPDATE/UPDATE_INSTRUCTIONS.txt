================================================================================
                    GSMT INDICES TRACKER - UPDATE PATCH
                              Version 8.1.3
================================================================================

This update fixes all issues with the Indices Tracker:
  ✓ All markets on ONE unified graph (no more infinite scroll)
  ✓ Calendar/date picker restored
  ✓ Fixed height display
  ✓ Proper data plotting
  ✓ Markets only show during trading hours
  ✓ AEST/AEDT timezone support

FILES INCLUDED:
---------------
1. indices_tracker_unified.html - New unified tracker
2. unified_chart_config.js      - Chart configuration
3. UPDATE_INDICES.cmd           - Automatic updater
4. UPDATE_INSTRUCTIONS.txt      - This file

AUTOMATIC INSTALLATION:
-----------------------
1. Extract this update to your GSMT installation folder
2. Copy files:
   - indices_tracker_unified.html → frontend/indices_tracker_unified.html
   - unified_chart_config.js → frontend/js/unified_chart_config.js
3. Double-click UPDATE_INDICES.cmd
4. The new tracker will open automatically

MANUAL INSTALLATION:
--------------------
1. Navigate to your GSMT installation folder
2. Copy indices_tracker_unified.html to the frontend/ folder
3. Copy unified_chart_config.js to the frontend/js/ folder
4. Open indices_tracker_unified.html in your browser

HOW TO USE THE UPDATED TRACKER:
--------------------------------
• Click market buttons to select/deselect (e.g., ASX, Nikkei, FTSE, S&P 500)
• All selected markets appear on ONE graph
• Use the calendar to select historical dates
• Toggle between AEST and AEDT with the switch
• Select time period: 1 Day, 5 Days, 1 Month, or 3 Months

KEY FEATURES:
-------------
• UNIFIED VIEW: All markets on one chart (no more separate charts)
• NO SCROLLING: Fixed height container
• CALENDAR: Date picker for historical data
• TRADING HOURS: Markets only show data when open
• REAL DATA: Connects to backend servers when running

MARKET HOURS (AEST):
--------------------
• ASX All Ordinaries: 10:00 - 16:00
• Nikkei 225:        10:00 - 16:00
• Hang Seng:         11:30 - 18:00
• FTSE 100:          17:00 - 01:30
• DAX:               17:00 - 01:30
• S&P 500:           23:30 - 06:00

TROUBLESHOOTING:
----------------
If no data appears:
1. Ensure backend servers are running (ports 8000 and 8001)
2. Check internet connection
3. Markets may be closed (shows blank during off hours)

If calendar doesn't work:
- Clear browser cache and reload the page

If update doesn't apply:
- Manually copy the files as described above
