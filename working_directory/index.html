Backend Information

API Endpoints

FastAPI backend running on port 8002:

  • /api/quote/{symbol} - Current quote
  • /api/historical/{symbol} - Historical data
  • /api/markets - Multiple markets
  • /api/search - Symbol search

Response Format

Historical endpoint returns nested format: 

{
  "symbol": "^AORD",
  "period": "5d",
  "data": [...],  // Array is nested
  "dataPoints": 100
