## 🐛 TROUBLESHOOTING

### **ML Signals Panel Not Showing:**
**Symptom:** Panel missing or empty

**Solution:**
1. Check if trading has started (click "Start Trading")
2. Wait for first analysis cycle (~5 seconds)
3. Verify ML integration available (check logs)
4. Hard refresh browser (Ctrl+Shift+R)

### **No Decisions Appearing:**
**Symptom:** Decisions feed shows "Monitoring markets..."

**Solution:**
1. Ensure markets are open (check Market Status panel)
2. Wait for ML analysis to complete
3. Check if symbols are valid
4. Review logs for errors

### **Component Signals All Neutral:**
**Symptom:** All components show ⚪ NEUTRAL (---)

**Solution:**
1. This is normal on first startup
2. Wait for first market data fetch
3. Components update after data received
4. May take 5-10 seconds after Start Trading

### **Import Errors:**
**Symptom:** ModuleNotFoundError

**Solution:**
```bash
pip install pandas numpy pytz dash plotly yfinance
```
