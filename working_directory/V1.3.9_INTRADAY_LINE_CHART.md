## 🔧 Technical Implementation

### Key Code Changes

#### 1. Data Fetching (15-minute intervals)
```python
# Fetch 1 day of data with 15-minute intervals
ticker = yf.Ticker(symbol)
hist = ticker.history(period='1d', interval='15m')
```

**Why 15 minutes:**
- Matches reference image granularity
- Good balance between detail and performance
- Typical for intraday charts

#### 2. Percentage Calculation (from market open)
```python
# Calculate percentage change from first data point
first_price = hist['Close'].iloc[0]
pct_changes = []
times = []

for idx, row in hist.iterrows():
    pct_change = ((row['Close'] - first_price) / first_price) * 100
    pct_changes.append(pct_change)
    times.append(idx)
```

**Why from open:**
- Standard for intraday charts
- Shows how market progressed during the day
- Easy to compare relative performance

#### 3. Line Plot (Plotly Scatter)
```python
fig.add_trace(go.Scatter(
    x=times,
    y=pct_changes,
    mode='lines',
    name=info['name'],
    line=dict(
        color=info['color'],
        width=2
    ),
    hovertemplate=(
        f"<b>{info['name']}</b><br>"
        "Time: %{x|%I:%M %p}<br>"
        "Change: %{y:.2f}%<br>"
        "<extra></extra>"
    )
))
```

#### 4. X-Axis Time Formatting
```python
xaxis={
    'showgrid': False,
    'showline': True,
    'linecolor': '#444',
    'tickfont': {'size': 10, 'color': '#999'},
    'tickformat': '%I%p',  # 10am, 11am, 12pm, etc.
    'dtick': 3600000,      # Tick every hour (milliseconds)
}
```

#### 5. Y-Axis Configuration
```python
yaxis={
    'showgrid': True,
    'gridcolor': '#333',
    'zeroline': True,
    'zerolinecolor': '#666',
    'ticksuffix': '%',     # Add % to labels
    'side': 'right'        # Right-side axis
}
```
