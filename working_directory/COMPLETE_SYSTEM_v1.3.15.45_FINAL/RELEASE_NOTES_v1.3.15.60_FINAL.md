## 🐛 TROUBLESHOOTING

### **Issue: Dependency Installation Fails**
**Symptoms:**
```
[X] Failed to install keras/torch/transformers
```

**Solution:**
```batch
1. Check internet connection
2. Verify disk space (~2.5GB free)
3. Try manual install:
   cd C:\Users\[YOUR_NAME]\Regime_trading\COMPLETE_SYSTEM_v1.3.15.45_FINAL
   venv\Scripts\pip install keras torch transformers scikit-learn --index-url https://download.pytorch.org/whl/cpu
4. Restart launcher
```

### **Issue: Dashboard Won't Start**
**Symptoms:**
```
[X] Dashboard failed to start
ModuleNotFoundError: No module named 'dash'
```

**Solution:**
```batch
1. Install missing module:
   venv\Scripts\pip install dash plotly
2. Restart launcher
3. Select Option 1 again
```

### **Issue: LSTM Still Shows Fallback**
**Symptoms:**
```
WARNING: Keras/PyTorch not available
```

**Solution:**
```batch
1. Close all terminals
2. Open new terminal
3. Run launcher again (KERAS_BACKEND needs new session)
4. Verify output shows: [OK] Keras LSTM available (PyTorch backend)
```

### **Issue: FinBERT Timeout Errors**
**Symptoms:**
```
ReadTimeout: HuggingFace hub timeout
```

**Solution:**
```
This is normal on first run. The model is downloading.
Wait 2-5 minutes. Subsequent runs use cached model.
```
