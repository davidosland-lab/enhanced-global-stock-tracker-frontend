## 🎓 TECHNICAL DETAILS

### **Auto-Dependency Logic**
```batch
1. Detect environment (venv vs system Python)
2. For each dependency (scikit-learn, Keras, PyTorch, transformers):
   - Check if installed: pip show [package]
   - If missing: pip install [package]
   - Track installations
3. Configure KERAS_BACKEND=torch (session + persistent)
4. Proceed to main menu
```

### **Environment Configuration**
- **Virtual Environment:** Auto-detected via venv\Scripts\pip.exe
- **Python Commands:** Auto-set (venv or system)
- **KERAS_BACKEND:** Set to 'torch' (session + setx)
- **Pip Index:** PyTorch uses CPU index URL
