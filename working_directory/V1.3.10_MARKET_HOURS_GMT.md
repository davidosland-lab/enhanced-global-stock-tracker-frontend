## рҹ“Ҳ Data Flow

### Step 1: Fetch Data
```python
# Fetch 2 days to ensure full market coverage
hist = ticker.history(period='2d', interval='15m')
```

### Step 2: Convert to GMT
```python
hist.index = hist.index.tz_convert(pytz.timezone('GMT'))
```

### Step 3: Filter to Market Hours
```python
# Get latest trading day
latest_date = hist.index[-1].date()

# Apply market hours filter
mask = (
    (hist.index.date == latest_date) &
    (hist.index.hour >= market_open_hour) &
    (hist.index.hour <= market_close_hour)
)
market_hours_data = hist[mask]
```

### Step 4: Calculate % Change
```python
first_price = market_hours_data['Close'].iloc[0]  # Market open price
pct_change = ((close - first_price) / first_price) * 100
```

### Step 5: Plot with GMT Labels
```python
fig.add_trace(go.Scatter(
    x=times,  # Already in GMT
    y=pct_changes,
    mode='lines'
))
```

## рҹҢҚ Timezone Reference

### GMT (Greenwich Mean Time)
- **Standard:** UTC+0 (no daylight saving)
- **Used by:** UK (winter), West Africa, Iceland
- **Why GMT:** Universal reference, no DST complications

### Timezone Conversions
```
ASX (AEDT = GMT+11):
  10:00 AEDT = 23:00 GMT (previous day)
  16:00 AEDT = 05:00 GMT

US Markets (EST = GMT-5):
  9:30 EST = 14:30 GMT
  16:00 EST = 21:00 GMT

FTSE (GMT):
  8:00 GMT = 8:00 GMT (same)
  16:30 GMT = 16:30 GMT
```
