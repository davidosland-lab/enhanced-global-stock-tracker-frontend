TROUBLESHOOTING
===============

Issue: Still seeing signal errors
Solution: 
- Verify START.bat shows "v193.11.7.3" at startup
- Re-run INSTALL_COMPLETE.bat
- Check Python version: python --version (should be 3.12+)

Issue: Still trading after-hours
Solution:
- Check core/market_calendar.py exists
- Verify log shows "[CALENDAR] Trading only on open markets"
- If not, market_calendar module may not be loaded

Issue: Positions still changing randomly
Solution:
- Check for "[BLOCKED]" warnings (they should appear frequently)
- If positions STILL change without "[BLOCKED]" warnings:
  - Email logs to support
  - This indicates a different issue

Issue: Loop still crashes
Solution:
- Check Python version (must be 3.12+)
- Re-run INSTALL_COMPLETE.bat
- Check for disk space (need 5+ GB free)
- Review error logs for specific error message

Issue: Dashboard not loading
Solution:
- Check port 8050 is not in use: netstat -ano | findstr 8050
- Restart START.bat
- Try http://127.0.0.1:8050 instead of localhost
- Check Windows Firewall isn't blocking
