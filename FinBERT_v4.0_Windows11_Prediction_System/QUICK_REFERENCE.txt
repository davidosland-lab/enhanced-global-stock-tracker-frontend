üîó API ENDPOINTS
----------------
Predictions:
  GET  /api/predictions/AAPL           - Get cached prediction
  GET  /api/predictions/AAPL/history   - Prediction history (30 days)
  GET  /api/predictions/AAPL/accuracy  - Accuracy statistics
  POST /api/predictions/validate       - Manual validation
  GET  /api/predictions/scheduler/status - Scheduler info

Stock Data:
  GET  /api/stock/AAPL?period=1y       - Stock data + prediction
  GET  /api/sentiment/AAPL             - Sentiment analysis
  GET  /api/health                     - System health

üêõ TROUBLESHOOTING
------------------
Port in use:
  netstat -ano | findstr :5001
  taskkill /PID <PID> /F

Python not found:
  - Reinstall Python with "Add to PATH" checked
  - Or use: C:\Python311\python.exe app_finbert_v4_dev.py

Dependencies missing:
  pip install -r requirements.txt

Database locked:
  - Close all app instances
  - Delete trading.db-journal if exists
