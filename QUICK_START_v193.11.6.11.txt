⚠️ RISK MANAGEMENT

AUTOMATIC GATES:
----------------
✅ World Risk ≥ 80 → Blocks all trades
✅ US Market ≤ -1.5% → Blocks all trades
✅ VIX ≥ 30 → Requires 70%+ confidence
✅ Sentiment < 20 → Blocks negative gaps

POSITION SIZING:
----------------
Base × Confidence Factor × Gap Factor × Risk Multiplier

Example:
- Base: $10,000 (10% of $100k capital)
- Confidence: 81% → 1.0x
- Gap: 1.04% → 1.0x
- World Risk: 86.1 → 0.35x (extreme risk)
- Final Position: $10,000 × 1.0 × 1.0 × 0.35 = $3,500

SOLUTION: Run INSTALL_FIRST_TIME.bat (not "pip install flask" only!)

ISSUE: Gap predictions show 0.0%
SOLUTION: Run overnight pipelines first (RUN_US_PIPELINE.bat, etc.)

ISSUE: Dashboard won't start
SOLUTION: 
1. Check port: netstat -ano | findstr :8050
2. Kill process: taskkill /PID <pid> /F
3. Restart: START_DASHBOARD.bat

ISSUE: US gap prediction failing
SOLUTION:
1. Test: python pipelines/models/screening/spx_proxy_realtime.py
2. Check internet (needs Yahoo Finance)
3. Check futures available (ES=F, NQ=F, YM=F)
