THE UK GAP PREDICTION FIX (NEW in v193.11.7.4)
===============================================

PROBLEM IDENTIFIED:
File: pipelines/models/screening/ftse_proxy_realtime.py
Lines: 190-194

Code was using:
  futures_symbols = ['^FTSE']  # FTSE 100 Index

Why this is WRONG:
- ^FTSE = FTSE 100 INDEX (not futures)
- Only updates during market hours (08:00-16:30 GMT)
- Overnight pipeline runs BEFORE UK open
- Gets yesterday's close = STALE DATA
- Real futures trade 24/5 and reflect current sentiment

SOLUTION IMPLEMENTED:
1. Removed "Futures" component (15% weight)
2. Added "Asia" component (10% weight):
   - Nikkei 225 (^N225): 35%
   - Hang Seng (^HSI): 35%
   - ASX 200 (^AXJO): 30%
3. Redistributed weights:
   - Europe: 30% → 35% (+5%)
   - All other components remain same

Benefits:
✓ Fresh overnight data between Europe close and UK open
✓ No stale data issues
✓ Better correlation with FTSE movements
✓ Expected >70% directional accuracy
