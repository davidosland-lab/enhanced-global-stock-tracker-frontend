🐛 TROUBLESHOOTING
================================================================================

Problem: Can't start Web Control Center
Solution: pip install flask flask-cors

Problem: Port 5000 already in use
Solution: Edit web_control_center.py, change port to 5001

Problem: Dashboard iframe blank
Solution: Wait 30 seconds for startup, check status badge

Problem: Can't access from phone
Solution: Check firewall (allow port 5000), use PC's IP not "localhost"

Problem: Logs not streaming
Solution: Refresh page (F5), ensure service is running
