TROUBLESHOOTING
===============

PROBLEM: "Python not found" or "pip not found"
SOLUTION: Install Python 3.8+ from https://www.python.org/
          Make sure to check "Add Python to PATH" during install

PROBLEM: VERIFY_INSTALLATION shows errors
SOLUTION: Check VERIFICATION_ERRORS_TROUBLESHOOTING.md
          Most common: missing packages, wrong Python version

PROBLEM: Still getting import errors
SOLUTION: 1. Check finbert_v4.4.4/__init__.py exists
          2. Check finbert_v4.4.4/models/__init__.py exists
          3. Re-extract v1.3.15 ZIP completely
          4. Delete venv folder and re-run INSTALL.bat

PROBLEM: Sentiment still showing 0 articles for all stocks
SOLUTION: 1. Check internet connection
          2. Verify finbert_v4.4.4/models/__init__.py exists
          3. Some stocks genuinely have no news (normal for SOME stocks)

PROBLEM: LSTM training still failing
SOLUTION: 1. pip show torch (verify installed)
          2. pip show tensorflow (verify installed)
          3. Check logs for specific error messages
